# TaskMaster

## Objetivo

Sistema Web de gerenciamento de tarefas desenvolvido para a disciplina de Programação Web.

## Tecnologias Utilizadas

### Frontend

* Vue.js
* Vue Router
* Pinia
* Axios

### Backend

* Node.js
* Express

### Banco de Dados

* Supabase

## Funcionalidades

* Login de usuários
* Controle de sessão com Pinia
* Cadastro de tarefas
* Listagem de tarefas
* Edição de tarefas
* Exclusão de tarefas

## Estrutura do Projeto

TaskMaster

frontend

* views
* stores
* router
* services
* components

backend

* routes
* middleware
* lib

## Como Executar

### Backend

npm install

npm run dev

### Frontend

npm install

npm run dev
